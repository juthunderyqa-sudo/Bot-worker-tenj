# Особистий асистент — stable lite

Це спрощена стабільна версія бота.

Що працює без AI-залежності:
- задачі
- Google Calendar для задач з датою/часом
- список задач
- виконання задач на сьогодні
- бюджет
- покупки / продукти
- нотатки
- звіт за тиждень
- місця поруч через прямий Google Maps link

Що лишається через Gemini:
- короткі голосові до 30 секунд
- пошук

Якщо голосове або пошук не спрацювали, основні функції бота все одно не повинні зависати.

Після деплою відкрий:
/setup?key=ТВОЄ_ЗНАЧЕННЯ_SETUP_KEY
